1. Install-WindowsFeature -Name AD-Domain-Services -IncludeManagementTools
2. Install-ADDSForest -DomainName "sirshanova.com"
3. Wait for machine to restart
4. Open PS and run Set-MpPreference -DisableRealtimeMonitoring $true
5. Download https://github.com/davidprowe/BadBlood and extract BadBlood to C:\BadBlood
6. Run Get-ChildItem -Path C:\BadBlood -Recurse | Unblock-File
7. Run powershell.exe -ExecutionPolicy Bypass -File C:\BadBlood\Invoke-BadBlood.ps1
8. Run powershell.exe -ExecutionPolicy Bypass -File C:\BadBlood\Invoke-BadBloodGPO.ps1
8. Once BadBlood is complete, run BadBloodAnswerKey.ps1 to generate the answer key files for what is grossly misconfigured.
